✦ CAR-T · Lentiviral Integration-Site Pipeline

Built to tell intended CAR-cassette integration apart from vector-backbone integration

A genome-wide WGS pipeline that goes from the vendor's raw FASTQ to safety-graded integration sites. Every step earns its place — here's the goal it serves and the edge it has over conventional workflows.

Stage 1 · Raw reads → analysis-ready
Start from vendor raw FASTQ, end to end
Own the whole analysis from the company's raw data, with quality control on raw and trimmed reads.
↑ Goal: Full QC ownership from source data
↑ Edge: Duplicate-aware — PCR copies never inflate clone size
🧬
Stage 2 · One combined reference
Host genome and vector aligned together
Read the exact points where vector sequence joins the genome.
↑ Goal: Capture precise vector–genome junctions
↑ Edge: Single alignment — no separate vector step, junction bases preserved
📍
Stage 3 · Integration-site calling
Two independent lines of evidence
Pinpoint each integration site at single-base resolution.
↑ Goal: Single-base resolution on every site
↑ Edge: Split-reads and read-pairs cross-check every call — each site has a confidence tier
📊
Stage 4 · Clonal abundance
How large is each integrated clone
Estimate the relative size of each clone in the product.
↑ Goal: Accurate clone-size estimation across the product
↑ Edge: Counts unique shear sites, not raw reads — stays honest under PCR duplication
✂️
Stage 5 · Insert sizing & filtering
Keep substantial inserts, drop fragments
Separate meaningful integrations from short, fragmentary vector pieces.
↑ Goal: Separate real integrations from noise
↑ Edge: Insert length measured per event, filtered at a configurable threshold (e.g. >200 bp)
⚑
Stage 6 · Vector content  Signature Step
Intended payload vs. plasmid backbone
Show which vector regions integrated at each site — promoter, CAR cassette, LTRs, WPRE — and whether backbone came along.
↑ Goal: Know exactly what integrated, not just where
↑ Edge: LTR-bounded rule flags ori, AmpR, and bacterial backbone — invisible to conventional pipelines
